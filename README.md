## 🚀 Starting

Welcome to another fullstack project! This application consists of a social network of animal adoptions. It is possible to create a profile, NGO, follow or like users and NGOs.

## 📋 Prerequisites

* React
* Redux
* Axios
* Node
* Express 
* MongoDB
* Bcrypt
